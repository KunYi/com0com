## History

The original project was hosted on SourceForge and maintained in **CVS**.
As SourceForge has discontinued CVS repositories, the full CVS history
was converted to Git in 2025 to prevent loss of historical data.

The CVS commit history, including timestamps, branches, and tags,
has been preserved as faithfully as possible.
